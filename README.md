# Buscador de Bibliografía Científica Avanzado / Advanced Scientific Bibliography Search Engine 🔬

🇪🇸 **[ES]** Herramienta automatizada para la búsqueda y extracción de resúmenes (abstracts) de artículos científicos, utilizando inteligencia artificial (Groq API) y técnicas de Web/PDF Scraping.  
🇺🇸 **[EN]** Automated tool for searching and extracting abstracts from scientific articles, using artificial intelligence (Groq API) and Web/PDF Scraping techniques.

**👨‍🔬 Autor / Author:** Dr. Utiel Zagada Dominguez

---

## 📖 Descripción / Description

🇪🇸 **Español:**  
Este proyecto permite a los investigadores y estudiantes generar palabras clave optimizadas mediante IA y buscar automáticamente literatura científica en bases de datos académicas (CrossRef, Semantic Scholar). Además, lee directamente metadatos de páginas web de revistas y documentos PDF para extraer la información. Al finalizar, genera un archivo Excel ordenado.

🇺🇸 **English:**  
This project allows researchers and students to generate optimized keywords using AI and automatically search for scientific literature in academic databases (CrossRef, Semantic Scholar). In addition, it directly reads metadata from journal webpages and PDF documents to extract information. Finally, it generates an organized Excel file.

## 🚀 ¿Cómo usarlo localmente? / How to use it locally?

🇪🇸 **Español:**  
1. Instalar las dependencias: `pip install -r requirements.txt`
2. Ejecutar la aplicación: `python app.py`
3. Abrir en el navegador: `http://127.0.0.1:5000`

🇺🇸 **English:**  
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Open in your browser: `http://127.0.0.1:5000`
